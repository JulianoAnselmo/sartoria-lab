# Sartoria Lab

Estúdio de desenvolvimento sob medida. Sites, apps e sistemas cortados pro seu negócio — não pro mercado em geral.

## Estrutura do repositório

```
sartoria-lab/
├── brand/                      Fundações da marca
│   ├── posicionamento.md         pilares, ICP, anti-pitch
│   ├── tom-de-voz.md             guia de escrita + vocabulário
│   ├── visual.md                 paleta, tipografia, logo
│   ├── mensagens.md              taglines, bios, copy, assinatura
│   ├── moodboard.md              eixos de referência visual
│   ├── dominio.md                disponibilidade + recomendação
│   └── logo/
│       ├── sartoria-lab-logo.svg       lockup horizontal
│       └── sartoria-lab-simbolo.svg    símbolo isolado
│
├── servicos/
│   └── catalogo.md             5 linhas de serviço + método
│
└── site/                       Fase 2 — projeto web (a construir)
```

## Decisões fechadas (Fase 1)

- **Paleta:** Sartoria Clássica — papel `#F5F2EC`, carvão `#1A1A1A`, giz `#D9CFC0`, linha vermelha `#A83232`
- **Tipografia:** serifa editorial (display) + sans geométrica (UI) + mono (código)
- **Logo:** chevron `< />` com costura tracejada — híbrido tech + ofício
- **Tagline oficial:** *Tecnologia sob medida.*
- **Domínios recomendados:** `sartorialab.com.br` (principal) + `sartorialab.dev` (secundário)
